# v1.0.0

Initial release of `maksim-yurash/ecommerce-suite`.

Included:
- Laravel package service provider.
- Configurable internal route prefixes.
- Models, migrations, factories and seeders for an abstract online shop.
- Web CRUD controllers and publishable Blade templates.
- API CRUD controllers with resources, collections and FormRequest validation.
- X-API-VERSION middleware.
- CurrencyRate and DeliveryCost facades.
- Publishable package tests.
